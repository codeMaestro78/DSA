# 6-Month DSA Roadmap — FAANG & Top Startup Ready

> **Goal:** Systematically cover all high-frequency DSA topics asked by Google, Meta, Amazon, Microsoft, Uber, and top startups (Stripe, Airbnb, Twitter/X, etc.)
>
> **Timeline:** 26 weeks | ~2–3 hours/day | ~5–7 LeetCode problems/week per topic
>
> **Strategy:** High-frequency topics first → Build intuition → Revisit weak areas in final weeks

Day 6 — Full mock (timed, no hints)
Day 7 — Rest / review notes / spaced repetition
